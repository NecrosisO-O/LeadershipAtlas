# 网站 Alpha 状态说明（历史基线）

本文件用于记录网站 Alpha 阶段的基线完成情况。当前网站已继续推进到 Beta 预发布阶段，因此本文件主要保留为历史参考。

## 一、Alpha 已完成内容

- 已建立 `website/` 前端工程，技术栈为 `React + Vite + TypeScript`
- 已建立网站专用结构化题库与数据生成脚本
- 已将 `82` 题转换为前端可消费的数据文件
- 已接入 `leader_profiles`、`item_dimension_mapping.json`、`signature_dimensions.json` 的网站侧生成资产
- 已完成网站侧 `matching` 模块，能够输出：
  - `综合最像`
  - `风格最像`
  - `理念最像`
- 已完成四段式主流程：
  - 欢迎页
  - 引导页
  - 答题页
  - 结果页
- 已实现五类题型：
  - 单端 Likert
  - 双端 Likert
  - 情境单选
  - 强制权衡
  - 点数分配
- 已实现关键交互规则：
  - 首次作答自动下一题
  - 返回修改不自动下一题
  - 风格/理念之间轻量过渡页
  - 最后一题手动进入结果
  - 未答完不能进入结果页
- 已完成一版 `社论档案感` 基础视觉系统
- 已通过 `vite build` 构建校验
- 已完成网站侧最小烟测：
  - 题库数量为 `82`
  - 五类题型均成功生成
  - 抽测人物画像自喂时，可回归到自身为主结果
