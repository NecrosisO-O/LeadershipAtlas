# 匹配规则测试总结

## 一、总体结论

- 第一阶段（17人自我回归）结果：强通过 17 / 17，可接受 0 / 17。
- 第二阶段（人工构造答卷）结果：强通过 2，可接受 1，不通过 7。

- 第三阶段相邻人物区分测试显示：使用完整人物答卷时，多数组合可以区分，但在人工提取鲜明特征组合时，若干高相似人物簇仍会互相吸附。

## 二、当前判断

- 当前匹配规则与82题题单组合后，较之前版本有所改善。
- 完整答卷场景下系统整体稳定，且自我回归结果较强。
- 但在人工构造的鲜明特征场景下，高相似人物簇内部的区分仍不够稳定。

## 三、当前结论口径

- 82题版本相较79题版本进一步改善了部分人物簇的区分。
- 当前系统可按 `完整答卷场景可用、稀疏鲜明特征场景仍不够稳` 理解。
- 因此，当前规则应视为 `阶段性可用版本`，而不应表述为所有场景下都稳定的终版。
