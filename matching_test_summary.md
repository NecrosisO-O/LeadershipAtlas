# 匹配规则测试总结

## 一、总体结论

- 第一阶段（17人自我回归）全部通过，说明当前人物画像库与匹配计算在“自己匹配自己”层面是稳定的。
- 第二阶段无论是原始版还是补充版，人工构造答卷测试都暴露出同一个问题：当前匹配规则对“高相似人物簇”的区分不够稳定。
- 第三阶段相邻人物区分测试显示：使用完整人物答卷时，多数组合可以区分；但当只提取“鲜明特征组合”时，若干人物簇仍会互相吸附。

## 二、阶段性判断

- 第一阶段：强通过 `17 / 17`。
- 第二阶段（修正版）：强通过 `5`，可接受 `1`，不通过 `4`。
- 第二阶段补充测试：强通过 `0`，可接受 `1`，不通过 `5`。
- 第三阶段：普京/金正恩、斯大林/毛泽东、李光耀/朴正熙、丘吉尔/戴高乐、罗斯福/曼德拉均可区分，但部分组合分差较窄。

## 三、当前最重要结论

- 当前匹配规则 `可以用于完整人物答卷的自我回归与近邻区分`，但 `还不适合直接投入产品化使用`。
- 主要问题不在“人物库无效”，而在于：
  - 若只给出一组鲜明特征，系统会把若干高相似人物簇吸附到一起。
- 当前最明显的高相似人物簇包括：
  - `斯大林 / 希特勒 / 金正恩 / 蒙博托`
  - `李光耀 / 朴正熙 / 邓小平`
  - `毛泽东 / 卡斯特罗`
- 第二阶段还证明：
  - 人工构造答卷必须采用完整输入，不能用大面积缺失的稀疏答卷；
  - 但即便采用完整底稿加关键特征覆盖，若干人物簇之间的误吸附仍然存在，说明问题不是单纯由输入稀疏造成的。

## 四、当前建议

- 不建议现在直接上线或进入下一步产品化发布。
- 建议优先回到规则层或画像层，进一步处理高相似人物簇的区分问题。
