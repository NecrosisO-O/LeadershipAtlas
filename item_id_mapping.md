# 题号映射表

本文档用于建立三套编号之间的对应关系：

- `display_no`：前台显示题号（连续编号 1-73）
- `item_id`：后台稳定题号
- `legacy_id`：原有题号，用于追踪历史文件与审查记录

说明：

- 当前不改变题目顺序，只重排编号。
- 后台稳定题号采用：
  - 领导人风格题：`L01-L32`
  - 执政理念题：`G01-G41`
