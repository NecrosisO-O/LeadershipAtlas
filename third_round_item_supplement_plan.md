# 第三轮补题方案

本文件记录在第二轮补题后，仍然没有完全解决的高相似人物簇问题，以及建议继续补入的分叉题。

当前重点只针对以下剩余问题：

- 高压强人簇：`斯大林 / 希特勒 / 金正恩 / 蒙博托`
- 发展主义威权型：`朴正熙`
- 主权竞争型边界：`普京 / 金正恩`

说明：

- 本轮补题依然不新增大维度，只在现有维度下补少量“簇内分叉题”。
- 当前目标不是进一步优化全部人物，而是优先拉开仍然反复互相吸附的人物。

---

## 一、高压强人簇

### 剩余问题

当前系统已经能把该簇识别为同一类高压人物，但仍不够稳定地区分：

- 斯大林的组织化国家机器 / 路线清洗型极权
- 希特勒的大众魅力煽动 / 群众共同体神话型极权
- 金正恩的世袭领袖化 / 封闭共同体 / 极端主权安全
- 蒙博托的 patronage / 私人网络 / 掠夺型个人统治

### 建议补题 1

- 放置维度：`11 政治斡旋与资源整合方式`
- 题型：`单端 Likert`
- 题目：
  - `一个统治者要想真正稳住局面，最重要的不是建立公开规则，而是确保关键资源和位置始终掌握在最忠诚的人手里。`
- 主要拉开：
  - `蒙博托` vs `斯大林 / 希特勒`
- 说明：
  - 这题比一般的“控制”题更贴近 patronage / 内圈分配逻辑。

### 建议补题 2

- 放置维度：`10 组织建制与统筹能力取向`
- 题型：`双端 Likert（7点）`
- 题目：
  - `请在以下两端之间选择最接近你观点的位置：`
  - `左端：统治真正可靠，主要靠最高领导人对亲信和核心圈子的个人掌控`
  - `右端：统治真正可靠，主要靠国家机器按统一路线长期稳定运转`
- 主要拉开：
  - `蒙博托` vs `斯大林`
  - 也有助于拉开 `金正恩`
- 说明：
  - 这是当前仍然最值得补的一道高压强人簇分叉题。

### 建议补题 3

- 放置维度：`19 共同体代表性与身份领导`
- 题型：`单端 Likert`
- 题目：
  - `领导人的地位，应该被看作一个国家或共同体历史延续的一部分，而不只是现实政治中的职位。`
- 主要拉开：
  - `金正恩` vs `普京 / 斯大林 / 希特勒 / 蒙博托`
- 说明：
  - 这题不是一般的共同体代表性，而更接近“历史继承—领袖正统性”的方向，适合拉开世袭领袖化人物。

### 建议补题 4

- 放置维度：`13 公共沟通与符号化表达`
- 题型：`单端 Likert`
- 题目：
  - `一个强有力的政治共同体，离不开反复的大规模集会、仪式和象征性展示。`
- 主要拉开：
  - `希特勒` vs `斯大林 / 蒙博托`
- 说明：
  - 这题更接近群众性仪式动员，而不是一般的公共表达。

## 二、发展主义威权型（朴正熙）

### 剩余问题

补题后：

- `李光耀式技术治理国家` 已经明显改善
- 但 `朴正熙式发展威权型` 仍然不够稳

说明当前题单虽能测到：

- 高控制
- 高绩效优先
- 高国家干预

但还不够直接测到：

- 发展目标压强
- 军式推进感
- 在国家起步阶段把政治压缩到发展之后的取向

### 建议补题 1

- 放置维度：`1 秩序与安全优先观`
- 题型：`单端 Likert`
- 题目：
  - `在国家发展起步阶段，压缩政治冲突和异议空间，是推动现代化必须付出的代价。`
- 主要拉开：
  - `朴正熙` vs `李光耀 / 邓小平`
- 说明：
  - 这题把“发展优先”与“政治压缩”明确绑在一起，更贴近朴正熙的威权发展主义。

### 建议补题 2

- 放置维度：`8 结构化推进与任务压强`
- 题型：`单端 Likert`
- 题目：
  - `国家建设最有效的方式，是把宏大目标拆成明确指标，然后一级一级压下去直到完成。`
- 主要拉开：
  - `朴正熙` vs `邓小平`
  - 同时也帮助和 `李光耀` 区分
- 说明：
  - 这题比一般的“重视时限”更接近国家发展工程的推进方式。

## 三、普京 / 金正恩边界

### 剩余问题

当前：

- 普京和金正恩在主权、安全、竞争、边界上仍然容易接近
- 已有题更擅长测到他们的共性，而不是差别

两人真正更不同的地方在于：

- `金正恩`：世袭领袖化、封闭共同体、政体正统性高度人格化
- `普京`：大国竞争、国家战略地位、现实主义安全国家

### 建议补题 1

- 放置维度：`22 主权与自主性观`
- 题型：`单端 Likert`
- 题目：
  - `一个国家在国际政治中，必须被承认为独立的大国力量中心，而不是融入由他国主导的秩序。`
- 主要拉开：
  - `普京` vs `金正恩`
- 说明：
  - 这题更接近“大国地位政治”，比一般主权自主更贴普京。

### 建议补题 2

- 放置维度：`19 共同体代表性与身份领导`
- 题型：`单端 Likert`
- 题目：
  - `领导人的地位不仅来自现实权力，也来自一种可以被继承和延续的历史正统性。`
- 主要拉开：
  - `金正恩` vs `普京`
- 说明：
  - 与高压强人簇中的“世袭领袖化”题方向一致，但这里专门用于拉开普京/金正恩边界。
